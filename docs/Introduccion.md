

# Definición del Proyecto

## Nombre 
Sistema de E-Comerce y Gestion de inventario para Tienda de Perfumes

## Objetivo 

Desarrollar una aplicacion Web que permita
- Vender Perfumes de forma Online
- Gestionar Inventario
- Gestionar tiendas fisicas 
- Administrar empleados 
- Controlar pedidos y pagos
- Generar reportes de ventas
