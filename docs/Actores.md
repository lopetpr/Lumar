# Actores del sistema
## 1. Cliente
Persona que compra e interactua con el sistema

### Puede 
- Registrarse 
- Iniciar Sesion 
- Ver productos 
- Agregar al carrito 
- Comprar 
- Ver pedidos


## 2. Administrador
Persona que controla todo el sistema


### Puede 
- Gestionar Productos 
- Gestionar Inventario 
- Gestionar Tiendas
- Gestionar Empleados 
- Ver Reportes
- Gestionar Pedidos


## 3. Empleado 
Trabajador de la tienda

### Puede 
- Ver inventario 
- Registrar ventas fisicas
- Actualizar Stock 
- Gestionar Pedidos
